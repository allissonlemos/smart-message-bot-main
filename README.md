# Smart Message Bot

Aplicativo para formatação de mensagens de produtos para WhatsApp.

## Estrutura do Projeto

```
smart-message-bot-main/
├── frontend/          # React + TypeScript + Vite
├── backend/           # Node.js + Express API
└── README.md
```

## Como executar

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Tecnologias

**Frontend:**
- React 18 + TypeScript
- Vite
- shadcn/ui + Tailwind CSS
- React Router DOM

**Backend:**
- Node.js + Express
- CORS + Helmet
- dotenv

## Funcionalidades

- Formatação automática de mensagens para WhatsApp
- Preview em tempo real
- Cópia para área de transferência
- API REST para formatação de mensagens
