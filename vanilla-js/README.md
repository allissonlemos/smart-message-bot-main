# Smart Message Bot - Versão JavaScript Puro

Versão simplificada do formatador WhatsApp usando apenas HTML, CSS e JavaScript puro.

## Como usar

1. **Abrir o arquivo**: Simplesmente abra o arquivo `index.html` no seu navegador
2. **Não precisa de servidor**: Funciona diretamente no navegador, sem instalação
3. **Sem dependências**: Não precisa de Node.js, npm ou qualquer framework

## Funcionalidades

✅ **Extração simulada**: Botão gerador com dados de exemplo  
✅ **Preview em tempo real**: Visualização da mensagem formatada  
✅ **Cópia para clipboard**: Botão para copiar mensagem  
✅ **Edição manual**: Campos editáveis para ajustar informações  
✅ **Preview de imagem**: Exibição da imagem do produto  
✅ **Interface responsiva**: Design adaptado para mobile  
✅ **Tema escuro**: Visual moderno e profissional  

## Estrutura

```
vanilla-js/
├── index.html    # Interface principal
├── style.css     # Estilos e tema escuro
├── script.js     # Lógica da aplicação
└── README.md     # Este arquivo
```

## Limitações

- **Extração simulada**: Usa dados mockados (não faz scraping real)
- **Sem persistência**: Dados não são salvos entre sessões
- **Funciona offline**: Não precisa de internet após carregar

## Vantagens

- **Zero configuração**: Abra e use
- **Leve e rápido**: Sem frameworks pesados
- **Compatível**: Funciona em qualquer navegador moderno
- **Portável**: Pode ser usado em qualquer dispositivo

Para usar, simplesmente abra o arquivo `index.html` no seu navegador!
